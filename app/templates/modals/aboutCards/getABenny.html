Whenever you Get a Benny, the server will run your data and return a Numerical Distribution and a Chi Squared Statistic. The response is then parsed into a Benny, which includes:
  • Highcharts Display - illustrates your data against the Benford distribution
  • Chi Squared Statistic - evaluates your result against the set limit of 8.0
  • Data Table - shows the raw distribution values
  • Inclusions Report - shows which data was included in the analysis
The computed Numerical Distribution will not include percentages less than one.
